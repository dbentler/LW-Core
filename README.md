# LW-Core
LW-Core is a plugin that utilizes the Spigot 1.8.8 API to fulfill the needs of the Administrative team and the players.
Contains built in:
- King of the Hill
- Warps
- Set Spawn
- Night Vision
- Staff Mode
- Ping
- Announcements
- /Wild
- In game staff management system
- Mod mode
- Staff chat(?)
And much more.
